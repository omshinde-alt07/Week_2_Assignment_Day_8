# Indian Income Tax Calculator (FY 2024–25)

This is a simple Python program that calculates income tax under the **New Tax Regime (FY 2024–25)**.

## Features

* Takes annual gross income as input
* Applies ₹75,000 standard deduction
* Calculates tax based on income slabs
* Displays total tax and effective tax rate

## Tax Slabs (New Regime)

* ₹0 – ₹3,00,000 → 0%
* ₹3,00,001 – ₹7,00,000 → 5%
* ₹7,00,001 – ₹10,00,000 → 10%
* ₹10,00,001 – ₹12,00,000 → 15%
* ₹12,00,001 – ₹15,00,000 → 20%
* Above ₹15,00,000 → 30%

Standard Deduction: ₹75,000

## How to Run

1. Make sure Python is installed.
2. Run the file:

```bash
python tax_calculator.py
```

3. Enter your annual income when prompted.

## Example

Input:

```
1000000
```

Output:

```
Gross Income: ₹10,00,000
Taxable Income: ₹9,25,000
Total Tax Payable: ₹42,500
Effective Tax Rate: 4.25%
```
